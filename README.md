# Water-Wheel-Generating-Station
First project/idea with STEM club!
## Big Goal:
  Make a mini generating station (AC power) utalizing water as a motive force
  Have some sort of back-up power supply with load shedding capability so if motive force dissapears its   not a blackout to black-start.
  Power a standard 120Vac household LED bulb.
## Mini-goals:
  -Output to a 7-segment display comfortably
  -Read voltage of battery
  -Safegard Battery by having emergency disconnecting capabilities both mechanically and electrically (software with trip points+ transistors)
  ### Voltage conversion
    AC -> DC (Generator -> Battery)
    DC -> AC (Battery -> Load)
    AC -> AC (Generator -> Load)
  -Use Battery/Generator output for field current 
  -Hand-make a single-phase generator
  -(*Maybe*) Incorperate Solar panels as a plan C blackstart meathod.
  -Experiment with different sizes/combinations of water wheels (Wide (bunch in paralell), Long (bunch in series), Big/Small)
